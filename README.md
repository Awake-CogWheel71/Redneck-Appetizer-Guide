**Lead Engineer:** Rich  
**Deployment Platform:** Jeep Tailgate / Radio Shack

This repository contains the Official Standard Operating Procedure (SOP) for high-utility, low-thermal-requirement snacking. Developed and field-tested in the high-humidity environment of Pascagoula, Mississippi.
